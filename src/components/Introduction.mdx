## Getting Started

### Basic Usage

\`\`\`html
<tabworthy-dates
  id="datepicker"
  label="Choose a date"
  placeholder='Try "tomorrow" or "next week"'
></tabworthy-dates>
\`\`\`

### Range Mode

\`\`\`html
<tabworthy-dates
  id="datepicker"
  range
  label="Choose a date range"
  placeholder='Try "June 8 to 12"'
></tabworthy-dates>
\`\`\`

### With Constraints

\`\`\`html
<tabworthy-dates
  id="datepicker"
  label="Choose a date"
  min-date="2026-03-01"
  max-date="2026-03-31"
></tabworthy-dates>
\`\`\`

## Available Props

- **id**: Unique identifier (required for accessibility)
- **value**: Current value (string for single date, array for range)
- **range**: Enable range mode
- **label**: Label text for the input field
- **placeholder**: Placeholder text
- **locale**: Locale for date formatting and parsing
- **disabled**: Disable the component
- **minDate**: Earliest allowed date (YYYY-MM-DD)
- **maxDate**: Latest allowed date (YYYY-MM-DD)
- **inline**: Keep calendar always visible
- **showQuickButtons**: Show quick date selection buttons
- **firstDayOfWeek**: Which day starts the week (0=Sunday, 1=Monday)

## Natural Language Examples

The datepicker understands many natural language inputs:

- Relative: "tomorrow", "yesterday", "in 5 days", "3 weeks ago"
- Named days: "next Monday", "last Friday"
- Specific dates: "March 15", "15/03/2026", "2026-03-15"
- Ranges: "March 10 to 15", "next Monday to Friday"

Supported for English, Japanese, French, Dutch, Russian, and Portuguese.
